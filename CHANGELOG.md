# NHibernate.ObservableCollections - Changelog


## 5.0.0 (2025-11-11)

Improvements:
- Added **`.NET 9.0`** support.

Breaking Changes:
- Removed **`.NET 7.0`** and **`.NET 6.0`** supports.
- Replaced **`IndexRange`** with **`Microsoft.Bcl.Memory`**.


## 4.0.0 (2024-11-12)

Improvements:
- Added **`.NET 8.0`**, **`.NET 7.0`**, and **`.NET 6.0`** supports.
